光标移动

  1. ctrl + a: 回到行首
  2. ctrl + e: 回到行尾
  3. ctrl + b： 光标向后1个字符（左）
  4. ctrl + f： 光标向前1个字符（右）
  5. alt + b: 光标向后移动1个字
  6. alt + f： 光标向前移动1个字
  7. ctrl + l： 清屏(clear)

助记：ctrl: 字符 alt: 字

历史命令

  1. ctrl + p: previous
  2. ctrl + n: next
  3. alt + <: first line
  4. alt + >: last line
  5. ctrl + r: 从当前后（历史）搜索
  6. ctrl + s: 从当前前（新的）搜索
  7. alt + .： 获取上一命令的最后一个参数

改变文本

  1. ctrl + v TAB : 插入tab
  2. ctrl + t: 交手当前字符和前一字符
  3. alt + t: 交手当前字和前一字
  4. alt + l: 当前单词小写
  5. alt + u: 当前单词大写

删除和粘贴

  1. ctrl + k: 删除光标到行尾的所有字符
  2. ctrl + u: 删除光标到行首的所有字符
  3. ctrl + d: 删除(光标)一个字符
  4. alt + d: 删除(光标)一个字
  5. backspce: 删除(往后)一个字符
  6. ctrl + w |alt + backspace: 删除(往后)一个字
  7. ctrl + y: 粘贴
  8. alt + y: 粘贴，往kill-ring前走

emacs基本操作

  1. 进入：emacs -nw (alias e='emacs -nw'; e)
  2. 退出：C-x C-c
  3. 保存文件：C-x C-s(save)
  4. 打开文件：C-x C-f(find)
  5. 只关闭当前文件： C-x k(kill)

屏幕与缓冲区

  1. 关闭其它屏： C-x 1
  2. 左右分屏：C-x 3
  3. 上下分屏：C-x 2
  4. 跳到其它屏： C-x o
  5. 列出缓冲区： C-x b
    1. 缓冲区下一文件：C-s
    2. 缓冲区上一文件：C-r
  6. 取消当前输入的命令: C-g(任何时候都有效)
